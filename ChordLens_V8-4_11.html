ChordLens — MIDI Chord Recognition

Real-time MIDI chord recognition, grand staff notation, and animated piano keyboard.

Connect a MIDI keyboard via USB. This app uses the Web MIDI API (supported in Chrome and Edge).

Sampled instruments use the MusyngKite SoundFont via soundfont-player — real recorded samples for acoustic grand piano, electric pianos, harpsichord, vibraphone, and more. Samples load on first use (requires internet). Synthesized presets use the Web Audio API oscillator engine and work offline.

Recognizes triads, 7ths, 9ths, 11ths, 13ths, altered dominants, slash chords, and inversions. Select a key signature to control enharmonic spelling.
